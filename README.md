# Luca Predieri — Personal Website

A minimalist, high-performance personal website inspired by [pradyunsg.me](https://pradyunsg.me/). This site serves as a professional portfolio, blog, and photo gallery.

## 🚀 Features

- **Responsive Design**: Fully functional across desktop, tablet, and mobile devices.
- **Dark Mode**: Automatic system preference detection with a manual toggle and persistent storage.
- **Photo Gallery**: A clean, title-free grid layout featuring an interactive lightbox for high-resolution viewing and detailed descriptions.
- **Blog Content**: Custom-styled article pages using high-legibility typography (`Lora`).
- **Professional Branding**: Custom 'LP' favicon and integrated social links.

## 🛠️ Tech Stack

- **Core**: Semantic HTML5 & Modern CSS3
- **Logic**: Vanilla JavaScript (no heavy frameworks)
- **Typography**: 
  - `Inter`: UI and headings
  - `Lora`: Long-form reading
- **Icons**: Hand-crafted SVG icons for social media and theme toggling.

## 📂 Project Structure

- `index.html`: Homepage with hero section and social links.
- `about.html`: Detailed professional timeline and biography.
- `blog.html`: Listing for articles and thoughts.
- `pictures.html`: Interactive photo gallery.
- `posts/`: Directory containing individual blog articles.
- `images/`: High-quality optimized assets.
- `style.css`: Core design system and global styles.
- `gallery.css` & `gallery.js`: Specialized styles and logic for the photo experience.
- `main.js`: Site-wide functionality (theme toggle, navigation).

## 🎨 Design Philosophy

The site prioritizes **visual excellence** and **typography**. It uses a curated, harmonious color palette that adapts to light and dark environments, ensuring a premium feel without the overhead of modern JavaScript libraries.
